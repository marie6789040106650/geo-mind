## 🚀 项目简介

**GEO Mind** 是一个专注于 **GEO (Generative Engine Optimization)** 的服务平台。我们致力于帮助企业在 AI 搜索时代（如 ChatGPT, Perplexity, Gemini 等）建立清晰的实体身份，通过结构化数据（Schema Markup）和语义增强内容，提升品牌在生成式引擎中的可见度和信任度。

**核心理念：**
*   **AI 认知深度**：先被 AI 认识，再被人看见。
*   **双向理解**：内容既要打动人，也要能被 AI 精准解析。
*   **建立信任**：用数据和结构化证据链，赢得算法的信任。

## 🛠 技术栈

本项目基于现代前端技术栈构建，追求极致的性能和 SEO 友好度：

*   **核心框架**: [React](https://react.dev/) (v18)
*   **构建工具**: [Vite](https://vitejs.dev/) - 极速的开发服务器和打包工具
*   **样式库**: [Tailwind CSS](https://tailwindcss.com/) - 原子化 CSS 框架，配合自定义动画
*   **图标库**: [Lucide React](https://lucide.dev/)
*   **语言**: [TypeScript](https://www.typescriptlang.org/)

## 💻 本地开发指南

1.  **克隆项目**

    ```bash
    git clone <your-repo-url>
    cd geo-mind
    ```

2.  **安装依赖**

    ```bash
    npm install
    ```

3.  **启动开发服务器**

    ```bash
    npm run dev
    ```

    打开浏览器访问 `http://localhost:3000` 即可预览。

4.  **构建生产版本**

    ```bash
    npm run build
    ```

## 🌐 部署

本项目已针对 **Vercel** 进行优化。

1.  将代码推送到 GitHub。
2.  在 Vercel 中导入项目。
3.  Vercel 会自动识别 Vite 配置并完成部署。
